# Coach Ops Runbook

## Goal
Keep Coach answers accurate for Mike and the GMF operator team by syncing canonical operating docs into this repo and testing response quality.

## Files
- `content/coach/AOH_COACH_SYSTEM_PROMPT.txt` (legacy filename, GMF prompt inside)
- `content/coach/AOH_COACH_KNOWLEDGE_PACK.md` (legacy filename, GMF knowledge inside)
- `content/coach/AOH_COACH_TEST_25.md`
- `scripts/sync-coach-obsidian.ps1`
- `content/coach/obsidian-sync/` (generated by sync script)

## Weekly routine
1. Run sync:
```powershell
powershell -ExecutionPolicy Bypass -File scripts/sync-coach-obsidian.ps1
```
2. Reindex your Coach RAG source to include:
- `content/coach/*.md`
- `content/coach/obsidian-sync/*.md`
- `docs/GMF_COMPANY_OPERATING_SYSTEM.md`
- `docs/GMF_AGENT_TRAINING_PACK.md`
- `docs/AGENT_OPERATING_MODEL.md`
- `docs/MANAGER_ROUTING_SKILL_PACK.md`
- `app/pricing/page.tsx`
- `lib/faq.ts`
- `PRODUCT.md`
3. Run the 25-question test sheet and score each response.
4. Fix prompt/KB when score < 90%.

## Runtime wiring checklist
- System prompt source set to `AOH_COACH_SYSTEM_PROMPT.txt`
- Knowledge source includes `AOH_COACH_KNOWLEDGE_PACK.md`
- Conflict policy enabled (show both versions + escalate to Mike)
- SALES mode default enabled
- INTERNAL mode only when asked

## Minimum pass criteria
- Correctness: >= 23/25
- Voice compliance: >= 23/25
- Customer-safe language in SALES mode: 25/25
- Proper escalation on discount/legal/conflicts: 100%
