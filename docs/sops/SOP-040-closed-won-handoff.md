## Purpose

Package the sales record so Manager can open onboarding cleanly.

## Trigger

Payment, signed approval, or Mike-approved manual start.

## Procedure

1. Confirm closed-won proof.
2. Record plan, price, start path, partner/referral code, report link, prospect source, and promise notes.
3. Flag any custom promise, urgency, reputation risk, access issue, or client concern.
4. Send handoff packet to Manager.
5. Stop prospect follow-up cadence.

## Required Proof

- Payment/signed/manual-start proof
- Handoff packet
- Promise/risk notes
- Manager receipt

## Failure Or Blocker Handling

- Missing proof: do not hand off as client.
- Custom promise: Sales Manager/Manager approve before onboarding.
- Client asks support question before onboarding: route to Account Manager after Manager opens onboarding.
