## Profile Setup Checklist

Confirm before creating or editing:

- Official business name to use publicly
- Website URL
- Primary business phone
- Public email/contact path
- Business category
- Service area vs physical address display
- Hours or appointment-only status
- Short business description
- Services list
- Logo
- Cover image/photos
- Verification method available
- Google account owner
- Manager access email for GMF operators, if separate from owner

## Setup Procedure

1. Check whether a GMF profile already exists.
   - Search Google Maps/Search for `GetMeFound`, `getmefound.ai`, and known phone/address variants.
   - Do not create a duplicate if a claimable profile exists.

2. Confirm business facts.
   - Use exact business name and current website/phone.
   - Decide service-area vs public address.
   - Confirm primary category and services.

3. Create or claim the profile.
   - Use the owner-controlled Google account.
   - Follow Google's verification flow.
   - Do not share passwords.

4. Add Manager access if needed.
   - Add the GMF operator email as Manager only after owner/control is clear.

5. Complete foundation fields.
   - Name, category, service area/address, website, phone, hours, description, services, logo/photos.

6. Capture proof.
   - Save setup/claim/verification status.
   - Save before/after profile screenshots once public.
   - Save profile URL.

7. Run internal Get Found audit.
   - GBP audit
   - Google AI Search readiness audit
   - website/profile fact sync
   - crawlability/index eligibility check
   - review link capture after available
