## Purpose

Classify prospect replies quickly and route each one to the right next action.

## Trigger

Prospect replies to cold outreach, report delivery, follow-up, partner/referral intro, or inbound sales email.

## Classifications

| Classification | Action |
|---|---|
| Report request | Open/report job under SOP 002 |
| Buying/pricing question | Use pricing/checkout SOP |
| Fit call request | Route to call request SOP |
| Objection | Use objection handling or Sales Manager review |
| Unsubscribe/DNC | Suppress immediately under SOP 021 |
| Complaint/legal/reputation risk | Stop and escalate to Manager/Auditor |
| No-fit | Mark no-fit and stop sales pressure |
| Nurture | Set allowed future touch |
| Client/support issue | Route to Account Manager/Manager, not sales |

## Procedure

1. Read reply for intent, risk, and urgency.
2. Classify with one primary status.
3. If unsubscribe or complaint, handle that first.
4. Assign next owner and due date.
5. Send only approved response type.
6. Log reply, classification, response, and next action.

## Required Proof

- Reply text/source
- Classification
- Response or route
- Suppression record if applicable
- Next owner
