# Profile Local Visibility Knowledge Pack

Status: draft v1
Last researched: 2026-05-16
Owner agent: Profile
Scope: Google Business Profile health, AI/local visibility, profile proof, and publishing coordination.

## Job

Profile owns the visible public presence of a local business, especially Google Business Profile.

Profile is not only an access agent. Profile should know how to inspect whether a business profile is complete, trustworthy, active, and useful for review automation and AI visibility.

## Profile Health Signals

For each client, Profile checks:

- business name
- address or service area
- phone
- website
- hours and holiday hours
- primary category
- secondary categories
- services/products
- business description
- photos/logo/cover
- reviews
- review replies
- Q&A
- posts/updates
- booking/order/menu/service links where relevant

## Review Signals

Profile should monitor:

- total review count
- average rating
- new reviews
- unanswered reviews
- last review date
- review velocity
- negative/low-star reviews
- review themes customers mention
- competitor review gap

For Review Automation, Profile hands the Google review link to GHL Expert.

For AI Visibility, Profile uses review themes and profile completeness as trust signals.

## Access And Verification

Profile owns:

- manager invite accepted
- correct location confirmed
- verification status checked
- duplicate/wrong location risks
- agency/business group organization if used
- review link captured

If a profile is not verified, Manager follows up with the client and GHL Expert continues non-dependent setup only.

## AI Visibility Checks

Profile should run periodic visibility checks:

- Google Maps/category search
- Google Search brand/category queries
- ChatGPT/Claude/Perplexity prompts where appropriate
- competitor comparison
- citation/NAP consistency
- recent review and profile activity

The goal is not to "game AI." The goal is to improve public proof so search and AI tools can confidently understand and recommend the business.

## Publishing Handoff

Profile owns what should be healthy on the GBP.

Press owns scheduled publishing once access is working.

HighLevel's Social Planner and GBP Post Scheduler can publish Google Business Profile posts. GHL Expert/Profile monitor connection health; Press manages approved content calendar and proof of publish.

## Monthly Profile Report

Profile should report:

- review count change
- average rating change
- new reviews
- unanswered reviews resolved
- profile fixes completed
- posts published
- visibility benchmark notes
- competitor gap
- recommended next actions
